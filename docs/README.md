## 🎯 目的別ガイド

### 初めてプロジェクトに参加する場合
1. [プロジェクト概要](../README.md) で全体像を把握
2. [引き継ぎガイド](./03_引き継ぎガイド.md) でプロジェクト情報を確認
3. [プロジェクト概要](../README.md) の「クイックスタート」でローカル環境を構築

### デプロイを実行する場合
1. [デプロイガイド](./01_デプロイガイド.md) でデプロイ方法を選択
2. GitHub Actions使用の場合は[ワークフロー解説](./02_ワークフロー解説.md)も参照

### ワークフローをカスタマイズする場合
1. [ワークフロー解説](./02_ワークフロー解説.md) で現在の仕組みを理解
2. `.github/workflows/deploy.yml` を編集

### トラブルシューティング
1. [デプロイガイド](./01_デプロイガイド.md) のトラブルシューティングセクション
2. [引き継ぎガイド](./03_引き継ぎガイド.md) の重要な注意事項
