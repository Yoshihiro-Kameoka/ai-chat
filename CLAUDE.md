# AIチャットボット仕様書

## プロジェクト概要
エンターテイメント目的のAIチャットボットWebアプリケーション

## 技術スタック
### フロントエンド
- **フレームワーク**: Next.js (App Router)
- **UIライブラリ**: React
- **スタイリング**: TailwindCSS（ビジネスライクなデザイン）
- **レスポンシブ対応**: 必須

### バックエンド
- **フレームワーク**: Next.js (App Router) + Hono
- **ORM**: Prisma.js
- **AIエージェント**: Mastra
- **AIモデル**: Claude-3-sonnet (Anthropic)
- **データベース**: MongoDB

### インフラストラクチャ
- **デプロイ先**: Google Cloud Run
- **想定同時接続数**: 5-10人

## 機能要件
### 必須機能
1. **AI対話機能**
　 - Claude-3-sonnetを使用した自然な対話
　 - リアルタイムレスポンス
　 - エラーハンドリング

2. **セッション管理**
　 - セッション中のみ会話履歴を保持
　 - ブラウザを閉じると会話履歴をクリア
　 - メッセージ数の制限なし

### 非機能要件
- **認証**: 不要
- **多言語対応**: 不要
- **特殊機能**: 不要（画像認識、音声入力など）
- **会話履歴検索**: 不要

## UI/UX設計
### デザイン要件
- **テーマ**: ビジネスライク
- **レスポンシブ**: PC/タブレット/スマートフォン対応（中略）
- @anthropic-ai/sdk
- tailwindcss
- typescript

## デプロイ手順

1. Google Cloud Projectの作成
2. Cloud Runの有効化
3. Dockerfileの作成
4. Cloud Buildでのイメージビルド
5. Cloud Runへのデプロイ
6. 環境変数の設定
